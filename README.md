# Online-Clothing-Store-System
# Overview

The Online Clothing Store Management System is a database-driven e-commerce application developed using C++ and MySQL. The system is designed to simplify the operation of an online clothing business by managing products, customers, orders, payments, inventory, and business reports.

The project aims to improve inventory management, automate payment calculations, and provide sales analysis for business decision-making.

# Technologies Used
- C++
- MySQL
- SQL Database
- Object-Oriented Programming (OOP)
